# Proyecto  - Modelo SIR con Streamlit

Universidad Nacional Mayor de San Marcos  
Facultad de Ciencias Matemáticas  
Curso: Técnicas de Modelamiento  

## Descripción
Aplicación web interactiva para simular la propagación de un rumor usando el modelo SIR adaptado a diferentes escenarios.
SIR para la propagación de la gripe porcina, rumor en el curso de derecho penal y porpagación de sectas ideologícas. 

## Ejecución
```bash
pip install -r requirements.txt
streamlit run app.py
